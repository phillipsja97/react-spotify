# Spotify Dashboard

## Description
* This application uses Spotify Web API to pull a spotify users profile and music history. Users are lead through a Node.js server which logs them into their spotify account. 

## Tehcnologies Used
* React.Js
* Node.Js Server
* Javascript
* Spotify Web API
* Spotify Widget
* Material UI
* React-Router-DOM
